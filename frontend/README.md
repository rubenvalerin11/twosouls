# TwoSouls Frontend (Next.js 14)

Frontend moderno para la tienda de ropa **TwoSouls**, conectado al backend en Node + MongoDB.

## Scripts

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
npm start
```

## Variables de entorno

Crear un archivo `.env.local` en la raíz:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
```

Cambia la URL por la de tu backend en Railway/Render cuando lo tengas en producción.
